행위 관련 디자인 패턴 (Behavioral) 
===========


### 특징  
- 객체나 클래스 사이의 알고리즘이나 책임 분배에 관련된 패턴
- 한 객체가 혼자 수행할 수 없는 작업을 여러개의 객체로 어떻게 분배하는지, 또 그렇게 하면서 **객체 사이의 결합도를 최소화**하는 것에 중점을 둔다
